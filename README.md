# 小米手环 NFC 自制表盘

这个仓库用于整理和维护小米手环 NFC 自制表盘资源。

## 下载表盘

成品 `.bin` 文件已放在 GitHub Releases 中，主分支不再保存可安装成品文件。

- Release：`v1.0.0`
- 下载页面：https://github.com/sep1107/miband-watchfaces/releases/tag/v1.0.0

| 文件 | 说明 |
| --- | --- |
| `miband7.bin` | 小米手环 7 NFC 自制表盘文件 |
| `miband9.bin` | 小米手环 9 NFC 兼容版表盘文件；与手环 7 同分辨率版本 |

## 源资源

`source/` 目录用于存放整理后的表盘配置与原始资源，方便后续继续适配其他设备。

## 使用说明

1. 打开 Releases 页面。
2. 下载对应设备的 `.bin` 文件。
3. 使用支持小米手环 NFC 表盘导入/替换的工具进行安装。
4. 安装前建议先备份原表盘文件。
